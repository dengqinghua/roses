2017-12-10

Author: 伍々慧, 1=bE, 试听: 这里

由 完整曲谱 中可以看到, 该曲的五线谱谱号为bbb, 查看五线谱的对照关系, 为 bE 调, 从维基百科 中可以知道: 该谱号约定为一个 # 表示添加一个5度, b 表示减一个5度.

下图为 Circle of Fifths 的完整谱号记录

参考: 五线谱调号的识别

bE调的音阶为

b3 4 5 b6 b7 1 2

对应的 C 小调 的音阶为

1 2 b3 4 5 b6 b7

但是该部分并未用的是 bE/c 相关的和弦, 因为加了变调夹夹在第三品, 所以使用的还是 C/a 相关的和弦, 其中有 a -> A 的转变, 所以用到了一些 A 大调(6 #7 #1 2 3 #4 #5 6)的中 #4, #5的一些音符.

1 INTRO-1

Fmaj7 -> Edim7 -> Dm7 -> E7 -> Am

级数走向

IVmaj7 -> IIIdim7 -> IIm7 -> III7 -> VIm7

该和弦也可以认为是 E7/B, 个人觉得音乐的魅力就在这里, 虽说看似都是简单的套路走向, 但是一点点变化音, 转位, 就会让音乐整体变得丰富.

3 Part A

Dm7 -> G7 -> C -> C -> Bdim7/D -> E -> Am
Dm7 -> G7 -> C -> C -> Bdim7 -> E -> Am

和弦形式不再展示, 但是比较有意思的是这几点:

3.1 bE

G7 的部分有一个 bE 的经过音

bE 不在 A大调的旋律音(6 #7 #1 2 3 #4 #5 6)中, 但是她作为一个经过音

G7              C
4 -> 3 -> b3 -> 3

该方式在 押尾桑的黄昏 也有应用

3.2 C

在C为根音的旋律为

6 #5 6 7 1 #1 2

该部分可以作为一个结束的旋律

Dm7 -> E7 -> E7(Solo) -> F, E

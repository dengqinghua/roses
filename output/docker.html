2 交互

2.1 启动, 监控和统计

下面的 guides 均为 container_name

docker run --name guides -i -t ubuntu /bin/bash


docker start/stop guides
docker attach guides # 进入容器

监控和统计

docker top guides
docker stats

i: interactive, t: pseudo-tty

docker run 的运行方式是什么样的? 如果生成多个 container 的时候, 会占用多少容量呢?

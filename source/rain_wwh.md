bE调的音阶为

```
b3 4 5 b6 b7 1 2
```

对应的 C 小调 的音阶为

```
1 2 b3 4 5 b6 b7
```

-------


```
IVmaj7 -> IIIdim7 -> IIm7 -> III7 -> VIm7
```

音阶
====

DATE: 2018-07-07

该文档涵盖了音阶的相关知识.

阅读完该文档之后, 您将了解到这些内容:

* 音阶小程序.
* 需要记住吉他1-12品上面的所有音.

--------------------------------------------------------

NOTE: 没有其他的聪明方法, 就记住吉他上每个品味是什么音即可.

Usage
-----
NOTE: 使用我的 [音阶练习脚本](https://github.com/dengqinghua/scales_practice)

### 测试音符
```shell
ruby guitar/practice.rb t  # 测试吉他
ruby ukulele/practice.rb t # 测试尤克里里

5 弦 第 8 品 是 什么音?

# 需要输入正确答案, 以吉他为例
4
```

### 测试品位位置
```shell
ruby guitar/practice.rb p  # 测试吉他
ruby ukulele/practice.rb p # 测试尤克里里

音 2 在 第 3 弦 是 第几品?

# 需要输入正确答案, 以吉他为例
7
```
